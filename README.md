# WSL Multiple Instance


Multiple instances of the same SO (Ubuntu OpenSuse Kali) on WSL

Export

```sh
wsl --export Ubuntu-20.04 .\Ubuntu20.tar
```

Import

```sh
wsl --import Ubuntu-20Dev .\Ubuntu-20Dev Ubuntu20.tar
```

view the instance on WSL

```sh
wsl -l -v
```
